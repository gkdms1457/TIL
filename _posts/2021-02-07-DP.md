 **입실론 - LDP**
 * 위의 DP와는 달리 Untrusted server 에서 정의
 * randomized response 매커니즘 이용  

    ```
    Q : 마약을 한적이 있나?

    A : 동전을 던져 앞면이 나오면 무조건 yes라 대답
                    뒷면이 나오면 사실대로 대답
    ==> 실제로 마약을 해서 yes라 대답했는지, 
            동전을 던져 앞면이 나와서 yes라 대답했는지 구분할 수 없음
    ```
