WebHacking 기초 - MySQL(2)

## INSERT - table에 row 추가
 `DESC tablename` - 테이블의 구조를 보여줌  
 **형식** 

 ```
 INSERT INTO (tablename) (column1, column2, ...) VALUES('value1', ...);
 ```

 **table내용 출력**  
 테이블의 모든 내용 출력

 ```
 SELECT * FROM (tablename) ;
 ```
